## ⬇️ **Installation**

1. Clone this repo

   HTTPS

   ```sh
   git clone https://github.com/dolbilov/BasicMVVMSample.git
   ```

   SSH

   ```sh
   git clone git@github.com:dolbilov/BasicMVVMSample.git
   ```

   GitHub CLI

   ```sh
   gh repo clone dolbilov/BasicMVVMSample
   ```

2. Restore dependencies

   ```sh
   dotnet restore
   ```

3. Run project
   ```sh
   dotnet run
   ```
